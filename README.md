# Financial Analytics Platform

A modular financial analytics platform designed for community banks.

## Current Modules
- Yield Statement
- Liquidity Dashboard (In Development)

## Planned Modules
- Budget Planning
- FTP Engine
- Allocation Engine
- Journal Generator
- Executive Dashboard
- Scenario Engine

## Architecture

The platform is built around reusable services:

- Hierarchy Engine
- Calculation Engine
- Data Engine
- Settings Engine
- Shared UI Components

Each module consumes these shared services.
